# Control Flow

## IF, Elif, Else Statement

```
if season == 'spring':
    print('Plant the garden')
elif season == 'summer':
    print('Water the garden')
elif season == 'fall':
    print('harvest the garden')
elif season == 'winter':
    print('Stay indoors')
else:
    print('unrecognized season')    
```
